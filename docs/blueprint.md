# **App Name**: StudioSuite

## Core Features:

- Dashboard: Provides an overview of key metrics like revenue, bookings, and low stock.
- Financial Analytics: Tracks revenue, expenses, and profit with visual charts.
- Product Management: CRUD operations for managing retail product inventory, including setting discounts and visibility.
- Service Management: CRUD operations for salon services, including descriptions and promotional pricing.
- Booking Management: Allows admin to manage bookings for all their stylists.
- Staff Management: Staff member profiles include roles, specialties, and performance metrics.
- Performance analytics: Uses booking statuses for performance tracking of services and individual team members

## Style Guidelines:

- Primary color: Bright yellow (#F5C518) for highlighting key elements.
- Background color: Dark grey (#080808) for a modern and sophisticated feel.
- Accent color: Purple (#7C3AED) to complement the primary yellow.
- Body and headline font: 'Inter' sans-serif font for clear UI readability
- Use 'lucide-react' icons throughout the interface for a consistent and modern look.
- Cards with rounded corners for grouping content, providing a clean and organized appearance.
- Subtle transitions and animations on hover for enhanced interactivity.
